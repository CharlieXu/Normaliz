Installation:

 

1. Download the basic package below.

2. Download one of the executable packages below.

3. Unzip both of them in a directory of your choice.

In this directory you will then find a directory Normaliz3.0 that contains the complete Normaliz system.

4. In the directory Normaliz3.0 you will find jNormaliz.jar. You can run jNormaliz by clicking it in the appropriate way on your desktop (requires Java).

5. If you want to run Normaliz or NmzIntegrate from the command line, it may be necessary to copy the executables normaliz(.exe) and nmzIntegrate(.exe) to a directory in your search path.

Executables (to be unzipped in the same directory):

 

Each package contains executables for Normaliz and NmzIntegrate.

 

Note: the statically compiled MacOS executable cannot handle exceptions properly. Therefore the automatic switch to infinite precision described in Section 4.4 of the Normaliz manual does not work. If you need infinite precision, please use --BigInt or compile Normaliz with shared libraries.
